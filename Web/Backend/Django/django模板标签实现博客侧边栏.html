Django 模板标签实现博客侧边栏

目标

博客侧边栏有四项内容：最新文章、归档、分类和标签云。这些内容相对比较固定，且在各个页面都会显示，如果像文章列表或者文章详情一样，从视图函数中获取然后传递给模板，则每个页面对应的视图 函数里都要写一段获取这些内容的代码，这会导致很多重复代码。更好的解决方案是直接在模板中获 取，为此，我们使用 Django 的一个技术：自定义模板标签来完成任务。

效果

为了实现上述功能，我们需要使用自定义模板标签，具体操作过程如下：

自定义模板标签

在我们的 article应用下创建一个 templatetags 包。包含 init.py 文件，之后在 templatetags\ 目录下创建一个 article_tags.py 文件，这个文件存放自定义的模板标签代码。

1. 最新文章模板标签

自定义模板标签代码写在 article_tags.py 文件中。其实模板标签本质上就是一个 Python 函数，因此按照Python 函数的思路来编写模板标签的代码就可以了.

@register.simple_tag
def get_recent_posts(num=5):
    # 这里的时间计算： 从元年到寻找创建的时间，从小到大，加上负号，就是从大到小，那么取前num个
    posts = ArticlePost.objects.all().order_by('-created_time')[:num]
    return posts

# 归档的模板标签
@register.simple_tag
def archives():
    # <QuerySet [datetime.date(2020, 3, 1)]>
    dates = ArticlePost.objects.dates('created', 'month', order='DESC')
    return dates

# 获取标签云
@register.simple_tag
def get_tags():
    # 数据库查看得到content_type_id=11,对应ArticlePost模型
    # counted_tags = TaggedItem.objects.filter(content_type_id=11)\
    counted_tags = TaggedItem.objects.filter(content_type__model=ArticlePost.__name__.lower())\
        .values('tag')\
        .annotate(count=Count('tag'))\
        .values_list('tag', 'count')
    counted_tags = dict(counted_tags)
    tags = TaggedItem.tag_model().objects.filter(pk__in=counted_tags.keys())
    for tag in tags:
        tag.count = counted_tags[tag.pk]
    return sorted(tags, key=lambda x: -x.count)

# 博客文章数据模型
class ArticlePost(models.Model):
    STATUS_CHOICES = (
        (0, '草稿'),
        (1, '发表'),
        (2, '更新'),
        (9, '删除'),
    )
    # 文章栏目的 “一对多” 外键
    column = models.ForeignKey(
        ArticleColumn,
        null=True,
        blank=True,
        on_delete=models.CASCADE,
        related_name='article',
        verbose_name='栏目'
    )
    author = models.ForeignKey(User, on_delete=models.CASCADE, verbose_name='作者')
    title = models.CharField(max_length=100, verbose_name='标题')
    ...

    # 内部类 class Meta 用于给 model 定义元数据
    class Meta:
        # ordering 指定模型返回的数据的排列顺序
        ordering = ('-created',)

    # 函数 __str__ 定义当调用对象的 str() 方法时的返回值内容
    def __str__(self):
        return self.title  # 将文章标题返回

    # 获取文章地址
    def get_absolute_url(self):
        return reverse('article:article_detail', args=[self.id])

文章归档后，可以点击进行年，月的删选， 最后将删选后的数据集渲染到文章列表中

def article_list(request):
    # 从 url 中提取查询参数
    search = request.GET.get('search')
    order = request.GET.get('order')
    column = request.GET.get('column')
    tag = request.GET.get('tag')
    year = request.GET.get('year')
    month = request.GET.get('month')
    # 初始化查询集
    article_list = ArticlePost.objects.all()
    # 搜索查询集
    if search:
        article_list = article_list.filter(
            Q(title__icontains=search) |
            Q(content__icontains=search)
        )
    else:
        search = ''
    # 栏目查询集
    if column is not None and column.isdigit():
        article_list = article_list.filter(column=column)
    # 标签查询集
    if tag and tag != 'None':
        article_list = article_list.filter(tags__name__in=[tag])
    # 时间查询集
    if year and year != 'None':
        article_list = article_list.filter(created__year=year).order_by('-created')
    if month and month != 'None':
        article_list = article_list.filter(created__month=month).order_by('-created')
