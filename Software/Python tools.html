python中itertools.islice的用法

itertools.islice的基本用法为： itertools.islice(iterable, start, stop[, step]) 可以返回从迭代器中的start位置到stop位置的元素。如果stop为None，则一直迭代到最后位置。

比如：

 from itertools import islice
 islice('ABCDEFG', 2)
1
2

返回一个迭代器：<itertools.islice at 0x108c29b88> 遍历迭代器会返回：A B

islice('ABCDEFG', 2, 4)
1

遍历会返回：C D

islice('ABCDEFG', 2, None)
1

遍历会返回: C D E F G

islice('ABCDEFG', 0, None, 2)
1

遍历会返回: A C E G

另外，如果在读取文件时也可以使用，比如不想读取文件第一行：

 with open('1.csv', 'r') as f:
    for line in islice(f, 1, None): 
        print(line)

from watermarker.marker import add_mark  # 导入watermarker

# 图片加水印
add_mark(file=r'./logo_deep-diary.png',         # 需要加水印的图片
         out=r"output_folder",                  # 输出目录

图片去除水印

去除水印后

代码详解

from PIL import Image
from itertools import product

# 图片去水印
img = Image.open('./output_folder/logo_deep-diary.png')     # 打开有水印的图片
width, height = img.size                                    # 得到图像的长宽
for pos in product(range(width), range(height)):            # 进行每个像素循环
    if sum(img.getpixel(pos)[:3]) > 0XD2*3:                 # 不同水印需要用不同逻辑
        img.putpixel(pos, (255,255,255))                    # 这里水印比logo更白，因此大于水印颜色的，把像素直接设置成白背景色
img.save('logo_deep-diary without watermarker.png')         # 保存去除水印后的图片声明：
