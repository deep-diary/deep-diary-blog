install

1.按 NPM 方式安装使用 ：

vue-awesome-swiper - npm (npmjs.com) (opens new window)

npm install swiper vue-awesome-swiper --save  // this will be wrong
npm install --save swiper  // this will be wrong
1
2

since both swiper and vue-awesome-swiper have many versions, so some of them will be uncompetitive, so we need to uninstall all of them and install the below version

npm install swiper@5.4.5 --save-dev
npm install vue-awesome-swiper@4.1.1 --save-dev

        <div slot="pagination" class="swiper-pagination"></div>
        <div slot="button-prev" class="swiper-button-prev"></div>
