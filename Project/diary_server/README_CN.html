说明

​ 本项目主要分为三大维度，第一维度是记忆追溯，是基于深度学习，融合日常视频，语音，图片，文本日记，聊天记录等记忆碎片，使用GCN训练成自己专属的记忆管家，平时您想不起来的事情，她可以帮您想起；第二维度是资源追溯，深入挖掘您的人脉资源，让每个您认识的人作为一个节点，每个节点都包含三大资源及三大需求，同样是基于图卷积网络，让您充分了解您身边朋友的需求及资源。第三维度是去中心化，基于区块链及web3概念，让您的数据更加安全，去中心化，让您的数据产生价值

项目简介

封面--一句话概括项目

突出产品独特亮点，用一句话总结产品核心概念

==使命：让个人数据产生价值，为自己，也为他人，使人们可以把握自己的数字资源==

==愿景：建立去中心化个人网盘平台，帮助用户低成本存储个人数据和社会资源，让其成为您终身相伴的记忆管家。==

==价值观：用户数据至上，诚信==

项目概况

痛点

  • 数据价值
    • 个人数据被企业获取并进行商业用途，商业利益却得不到分享
  • 存储隐私
    • 个人隐私数据得不到保护
    • 企业因为隐私以及信任问题，获取到的数据途径或者渠道非常有限
  • 数据内容
    • 非结构化数据难以融合搜索
    • 各种数据相对独立，不成体系
    • 个人记忆和社会资源相互独立

别人怎么做的

我们怎么做的

数据价值

1.企业使用用户数据获得的商业利润，将按照智能合约的方式，返给用户，作为数据使用费

2.平台上任何你发布的数据，别人使用过的，都可为您带来收益

存储隐私

1.将用户数据加密之后，存储在CESS

2.在采用数据可用不可见的技术背景下，用户敏感数据可以得到极大的保护

3.在个人授权下企业可以使用用户数据进行商业应用（如训练模型等）

数据内容

1.记忆追溯：充分提取视频，图片，音频，日记，gps轨迹，微信聊天记录等，并将其编码成结构化的向量

2.资源追溯：平台记录身边朋友的三大资源，三大需求，使用NLP编码成结构化向量

3.基于GCN融合所有记忆和资源，训练成为你的个人记忆

项目亮点

  1. 个人数据的价值挖掘：通过记忆追溯和资源追溯，将个人数据转化为具有商业和个人意义的价值。
  2. 去中心化和隐私保护：利用区块链和智能合约技术，实现个人数据的安全存储和隐私保护，赋予用户更多的控制权。
  3. 个性化记忆管家机器人：结合视觉功能和个性化互动，提供独特的家庭助手体验，满足用户的个性化需求和娱乐互动。

行业定位:

解决行业内个人数据管理和数据价值挖掘的问题

  • 高需求市场：记忆管家机器人可以满足人们对于个人记忆和回忆的渴望，提供个性化的回忆和交流体验。这在老年人、亲情关系维系、重要事件回顾等方面具有广泛的市场需求。
  • 家庭助手市场：随着智能家居技术的普及，家庭助手机器人具备了更多功能，记忆管家机器人作为其中的一种扩展，可以提供个性化的家庭助手服务，进一步满足用户的需求。
  • 社交和娱乐市场：记忆管家机器人具备人脸识别、目标识别等视觉功能，可以作为社交和娱乐工具。它可以帮助人们回忆过去的点点滴滴，并与朋友分享、互动，为社交活动增添乐趣。

行业前景:

  • 随着数字化生活的普及，个人数据管理和价值挖掘的需求日益增长。记忆追溯和资源追溯作为创新的概念，可以满足人们对于个人回忆、家庭互动和社交娱乐的渴望。同时，随着智能家居和人工智能技术的快速发展，记忆管家机器人具备广阔的市场前景。

  • 在数据驱动的市场环境中，用户数据归用户所有，将可以对企业对个人都有极大的益处，使得用户数据得到价值回馈，用户授权的情况下企业也可以获得更多的数据，赋予更多的商业用途

创新性

  1. 记忆追溯和资源追溯的结合，打破了传统数据管理和回忆方式，创造个人化的数字记忆管家。
  2. 利用区块链和智能合约技术，确保个人数据的安全性和隐私保护，赋予用户更多的控制权和价值回报机制。
  3. 结合记忆管家机器人的视觉功能和个性化互动，创新了社交娱乐领域，为用户带来更丰富的体验和连接。

这些问题如何解决的

**产品详解：**瞄准用户痛点，介绍过人优势，减少流行词使用；详细介绍使用步骤，避免陷入“流水账”；展现市场前景（验证产品具备应用市场和商业潜力）；使用表格样式展现各类调研分析方法和结果

深记，是一个旨在解决上述痛点的一个去中心化的平台

如果有这么一个平台，可以使用人脸识别技术，对照片视频出现过的人脸进行自动分类标注，可以使用目标识别技术，对照片中出现的事物进行有效识别并打上标签，可以使用图卷积网络技术，对所有记忆碎片记忆训练，得到真正属于你自己的智能记忆管家

记忆追溯

  • 去中心化存储和加密：利用区块链和去中心化存储技术，将个人数据安全地存储在分布式网络上，并通过加密技术保护数据隐私。
  • 区块链记录：使用区块链记录每个人的记忆追溯数据，确保数据的不可篡改性和可追溯性。
  • 智能合约奖励机制：通过智能合约实现记忆贡献者和使用者之间的奖励机制，鼓励个人贡献记忆数据并让其产生价值。
  • 数据隐私与加密: 利用密码学和区块链技术，保护个人数据隐私

这里的强化记忆，可以理解为记忆追溯，是基于多模态深度学习（Multimodality Fusion Technology，MFT），融合日常视频，语音，图片，文本日记，聊天记录等记忆碎片，使用GCN训练成自己专属的记忆管家，平时您想不起来的事情，她可以帮您想起；

资源追溯

  • 社交关系图：基于区块链技术构建一个社交关系图，每个人可以录入自己的三大资源和三大需求，并建立与其他人的连接关系。
  • 智能匹配算法：利用机器学习和图卷积网络等技术，开发智能匹配算法，将个人的资源与需求进行匹配，为个人提供个性化的推荐和合作机会。
  • 去中心化身份验证：使用区块链身份验证技术，确保每个人的身份真实可信，增强资源共享的信任度。
  • 同学会，校友会，忆往昔的同时，我们更倾向于了解别人的需求和资源抱团取暖
  • 聚会期间，使用该平台，录入简单的岗位信息同时，录入您的三大资源，三大需求，三大资源，可以帮您提高额外的收入，三大需求，可以利用平台的的资源，帮您对接解决您的需求。
  • 结合记忆追溯相关技术，再次对相关资源，需求，岗位信息进行融合，实现资源追溯。
  • 资源追溯，旨在于帮您充分挖潜您身边的资源和需求，实现抱团取暖
  • 社交关系起点: 以同学、校友为出发点，整合资源和需求
  • 个人资源与需求: 用户录入个人的三大资源和三大需求，构建资源追溯模型
  • 智能匹配算法: 利用机器学习和图卷积网络等技术，实现个性化的资源匹配和合作机会

去中心化

  • 使用区块链技术，让您的数据去中心化，更加安全，只属于您个人的，任何其它机构都无法操纵或者删除您的数据
  • 数据产生价值，深记平台永久免费，一些广告产生的收益，通过区块链技术，让您的数据给您带来收益
